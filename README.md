## Descripción general

Este proyecto consiste en la investigación de un perfil profesional soñado dentro del área de Ingeniería en Sistemas y en el desarrollo de un sitio web tipo portafolio profesional, utilizando **HTML5** y **CSS3**. El perfil seleccionado fue **Machine Learning Engineer / Applied AI Engineer**, debido a la alta demanda actual de profesionales capaces de diseñar, implementar e integrar soluciones de inteligencia artificial en entornos reales.

El análisis se basó en ofertas laborales reales publicadas en portales de empleo, identificando tecnologías, habilidades y responsabilidades comunes. A partir de ello, se definió el enfoque del sitio web, el cual estará orientado a la presentación de un perfil profesional enfocado en Machine Learning, IA aplicada y despliegue de soluciones inteligentes.

---

## Perfil profesional objetivo

### Machine Learning Engineer / Applied AI Engineer

El perfil profesional objetivo seleccionado es **Machine Learning Engineer / Applied AI Engineer**. Este perfil combina programación, modelado de datos, uso de frameworks de inteligencia artificial, integración de modelos en sistemas reales, despliegue en la nube y buenas prácticas de software.

A diferencia de un perfil exclusivamente orientado a investigación, este rol se enfoca en construir soluciones funcionales y escalables que generen impacto real dentro de productos y servicios.

## Análisis comparativo

Después de revisar las cinco ofertas laborales, se identificaron varios patrones comunes que permiten entender qué exige actualmente el mercado en puestos relacionados con Machine Learning e Inteligencia Artificial.

### Tecnologías más repetidas
- **Python** como lenguaje principal
- Frameworks de IA/ML como **PyTorch** y **TensorFlow**
- Herramientas para LLMs y NLP como **LangChain**, **LlamaIndex** y modelos generativos
- Plataformas cloud como **AWS**, **GCP** y **Azure**
- Uso de APIs, bases de datos y pipelines de datos
- Prácticas de **MLOps / LLMOps**

### Hard skills más comunes
- Entrenamiento, evaluación y ajuste de modelos
- Integración de soluciones de IA en sistemas reales
- Despliegue de modelos en producción
- Construcción de pipelines de datos y features
- Monitoreo, versionado y mantenimiento de modelos
- Uso de servicios cloud y herramientas de automatización

### Soft skills más comunes
- Comunicación efectiva
- Trabajo en equipo
- Resolución de problemas
- Adaptación al cambio
- Pensamiento crítico
- Aprendizaje continuo

### Responsabilidades más repetidas
- Diseñar e implementar soluciones de IA aplicadas a problemas reales
- Integrar modelos en productos y servicios
- Llevar soluciones desde la experimentación hasta producción
- Colaborar con equipos de negocio, producto y desarrollo
- Mejorar continuamente el rendimiento y confiabilidad de los modelos

---

## Conclusiones del análisis

Las ofertas analizadas muestran que el mercado actual no busca únicamente personas capaces de entrenar modelos, sino profesionales que puedan **integrar la inteligencia artificial en soluciones reales, escalables y útiles para el negocio**. Esto significa que el perfil más valioso combina conocimientos de machine learning, desarrollo de software, nube, APIs y buenas prácticas de despliegue.

También se observa que muchas vacantes valoran el trabajo con **LLMs, NLP, MLOps, automatización y sistemas orientados a producción**, lo cual confirma que la IA aplicada tiene una fuerte relación con la ingeniería de software moderna.

---

## Justificación del perfil profesional elegido

Se eligió el perfil de **Machine Learning Engineer / Applied AI Engineer** porque representa una combinación entre programación, inteligencia artificial y resolución de problemas reales. Este perfil permite trabajar en áreas como análisis predictivo, automatización, procesamiento de lenguaje natural, visión por computadora e integración de sistemas inteligentes dentro de aplicaciones modernas.

Además, este enfoque resulta coherente con las tendencias actuales del mercado y con el tipo de habilidades que las empresas más solicitan en sus ofertas laborales.

Aunque varias de las ofertas analizadas corresponden a puestos senior, fueron útiles para identificar las competencias más valoradas en el mercado y definir una ruta de crecimiento profesional desde etapas iniciales.

## Enfoque del sitio web

El sitio web será desarrollado como un **portafolio profesional** orientado a la **postulación ficticia** para un puesto como Machine Learning Engineer / Applied AI Engineer.

El objetivo del sitio será presentar:
- Perfil profesional
- Habilidades técnicas
- Proyectos relevantes
- Tecnologías dominadas
- Información de contacto

La idea es que el sitio funcione como una carta de presentación digital, con una apariencia moderna, clara y profesional, alineada con el perfil investigado.

---

## Diferenciadores clave

### 1. Base sólida en programación y pensamiento lógico-matemático
El perfil se apoya en una combinación fuerte de programación, razonamiento lógico y análisis estructurado, lo cual resulta esencial para comprender, entrenar e integrar modelos de machine learning.

### 2. Enfoque en soluciones prácticas
Más allá del interés por la inteligencia artificial como tema teórico, el enfoque está orientado a construir soluciones reales, útiles y funcionales, capaces de integrarse en sistemas productivos.

---

## Propuesta de estructura del sitio web

El portafolio web contará con las siguientes secciones principales:

1. **Inicio / Landing page**  
   Presentación breve, nombre profesional y enfoque principal.

2. **Sobre mí**  
   Descripción del perfil, intereses y objetivos profesionales.

3. **Habilidades**  
   Lenguajes, herramientas, frameworks y conocimientos relacionados con IA/ML.

4. **Proyectos**  
   Espacio para mostrar proyectos académicos o personales relacionados con programación, análisis o inteligencia artificial.

5. **Contacto**  
   Información básica de contacto y enlaces relevantes.
